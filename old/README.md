# 答题卡小题分匹配系统

将学生小题分叠加到扫描答题卡图片上，生成带分数的试卷图片，方便老师分析。

## 安装方法

### 1. 安装 Python（如已安装可跳过）

从 https://www.python.org/downloads/ 下载安装 Python 3.10 或更高版本。
安装时务必勾选 "Add Python to PATH"。

### 2. 下载本程序

将整个程序文件夹放到电脑上任意位置。

### 3. 安装依赖

双击运行 `安装依赖.bat`，或在程序文件夹中打开命令行执行：

```
pip install -r requirements.txt
```

### 4. 启动程序

双击运行 `启动程序.bat`，或在程序文件夹中打开命令行执行：

```
python app.py
```

启动后，浏览器会自动打开程序界面。如果没有自动打开，请手动访问 `http://localhost:5000`。

## 使用流程

程序分 5 个步骤引导完成操作：

### 第1步：上传小题分文件

- 点击或拖拽上传成绩文件（支持 `.csv`、`.xls`、`.xlsx`）
- 系统会显示数据预览，确认数据是否正确
- 配置数据列：
  - **忽略前几行**：跳过文件开头的非数据行（如标题行）
  - **考号所在列**：选择存放学生考号/学号的列
  - **满分所在行**：选择存放每题满分值的那一行（通常紧跟在表头后面）
  - **班级所在列**：可选，用于按班级分文件夹输出
  - **小题分起始列 / 结束列**：小题得分从哪列开始、到哪列结束

### 第2步：上传答题卡图片

- 点击或拖拽上传答题卡扫描件
- 支持的格式：`.jpg`、`.png`、`.bmp`、`.tiff`、`.pdf`
- PDF 文件会自动转换为图片（一页一张）
- 可以看到缩略图预览

### 第3步：标记分数打印位置

这是最关键的一步：
- 从下拉框选择一张答题卡图片作为模板
- **在图片上点击**每道小题分数要打印的位置
- 点击顺序对应小题顺序（第1次点击=第1题位置，第2次点击=第2题位置...）
- **右键**或点击"撤销"按钮可删除上一个标记
- 调整 **Y轴偏移** 微调所有标记的上下位置
- **双面试卷**：先标记正面，再切换到背面标记
- 可以将位置保存为**模板**，下次相同试卷直接加载使用

### 第4步：处理设置

- **试卷类型**：单面或双面（双面需在第3步分别标记正反面）
- **匹配方式**：
  - 条码识别：自动扫描答题卡上的条形码获取考号（推荐）
  - 文件名数字匹配：从文件名提取数字与考号匹配
- **字号大小**：印在答题卡上的分数文字大小，默认 80
- **按班级分文件夹**：勾选后输出结果按班级分文件夹存放

### 第5步：处理与下载

- 点击"开始处理"，等待处理完成
- 查看匹配结果：成功数量、未匹配的学生和图片
- 点击"下载处理结果"获取 ZIP 文件

## 输出说明

下载的 ZIP 文件包含：
- `已匹配/` - 成功匹配并打好分数的答题卡图片
- `无条码/` - 未能匹配到成绩的答题卡图片（需手动处理）

## 常见问题

**Q: 条码识别不工作？**
A: 确认安装了 pyzbar 库。Windows 用户可能需要安装 Visual C++ Redistributable。

**Q: PDF 文件无法上传？**
A: 确认已安装 PyMuPDF：`pip install pymupdf`

**Q: 字体显示乱码？**
A: 确保系统中有中文字体（如微软雅黑、黑体）。Windows 系统通常已自带。

**Q: 端口被占用？**
A: 可以用 `python app.py 5001` 指定其他端口启动。
